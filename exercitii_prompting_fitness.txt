
SET DE EXERCITII – PROMPTING PENTRU FITNESS (10 exercitii)
================================================================

INSTRUCTIUNI OBLIGATORII (pentru TOATE exercitiile):
Fiecare prompt scris de student TREBUIE sa contina explicit:
1. INSTRUCTIUNI clare si specifice pentru modelul llm
2. CONTEXT (situatia utilizatorului, constrangeri)
3. FORMAT DE IESIRE clar (structura, bullet points, pasi etc.)

Domeniu: FITNESS
Limbaj: LIMBA ROMANA
Nu se scrie cod, doar PROMPTURI.

----------------------------------------------------------------
1. ZERO-SHOT PROMPTING
----------------------------------------------------------------
Scrie un prompt zero-shot pentru generarea unui plan de antrenament de forta
pentru un adult sanatos, respectand TOATE instructiunile obligatorii.

Prompt:

Genereaza un plan de antrenament de forta pentru un adult sanatos.

Context utilizator:
	•	Varsta: 30 ani
	•	Nivel: incepator
	•	Disponibilitate: 3 antrenamente pe saptamana
	•	Acces la sala complet echipata
	•	Obiectiv: cresterea fortei si tonifiere musculara
	•	Fara accidentari sau probleme medicale

Instructiuni:
	•	Include exercitii pentru toate grupele musculare majore
	•	Specifica seturi, repetari si pauze
	•	Adauga recomandari de incalzire si stretching
	•	Foloseste explicatii clare si concise

Format de iesire:
	•	Titlu plan
	•	Recomandari generale (bullet points)
	•	Plan structurat pe Ziua 1 / Ziua 2 / Ziua 3
	•	Sfaturi finale pentru progres


----------------------------------------------------------------
2. ZERO-SHOT PROMPTING + PERSONIFICARE
----------------------------------------------------------------
Scrie un prompt zero-shot in care modelul este personificat ca
„antrenor personal certificat", respectand structura ceruta
pentru un antrenament de forta.

Prompt:

Actioneaza ca un antrenor personal certificat si creeaza un plan de antrenament de forta.

Context utilizator:
	•	Adult sanatos, 35 ani
	•	Nivel intermediar
	•	4 zile disponibile pentru antrenament
	•	Obiectiv: cresterea masei musculare

Instructiuni:
	•	Ofera explicatii motivationale si profesionale
	•	Include exercitii compuse si izolate
	•	Specifica intensitatea si timpul de recuperare

Format de iesire:
	•	Introducere motivationala
	•	Program pe zile (lista structurata)
	•	Recomandari nutritionale scurte
	•	Sfaturi de siguranta

----------------------------------------------------------------
3. FEW-SHOT PROMPTING
----------------------------------------------------------------
Creeaza un prompt care include 2 exemple (intrebare + raspuns),
apoi cere modelului sa raspunda la o a treia intrebare,
folosind context si format de iesire explicit (contextul: fitness).

Prompt:

Raspunde la cererea finala folosind stilul si structura din exemplele de mai jos.

Context general: fitness si antrenament pentru sanatate.

Exemplul 1:
Intrebare: Creeaza un mini-antrenament cardio pentru incepatori.
Raspuns:
	•	Incalzire: mers rapid 5 min
	•	Exercitii: jumping jacks 3x20, genuflexiuni 3x15
	•	Revenire: stretching 5 min

Exemplul 2:
Intrebare: Propune exercitii pentru abdomen acasa.
Raspuns:
	•	Plansa 3x30 sec
	•	Crunch 3x20
	•	Ridicari picioare 3x15

Cerinta finala:
Creeaza un antrenament pentru forta picioarelor la sala.

Format de iesire:
	•	Incalzire
	•	Exercitii (bullet points cu seturi si repetari)
	•	Stretching final



----------------------------------------------------------------
4. CHAIN OF THOUGHT
----------------------------------------------------------------
Scrie un prompt care cere modelului sa gandeasca pas cu pas
pentru a construi un plan de slabire, cu format explicit.


Prompt:

Construieste un plan de slabire explicand pas cu pas rationamentul.

Context utilizator:
	•	Adult 40 ani
	•	Sedentar
	•	Vrea sa slabeasca 8 kg in 4 luni
	•	Poate face sport 4 zile/saptamana

Instructiuni:
	•	Analizeaza obiectivul
	•	Determina tipul de antrenament optim
	•	Creeaza program progresiv

Format de iesire:
	1.	Analiza situatiei
	2.	Strategia aleasa
	3.	Plan saptamanal structurat
	4.	Recomandari nutritionale

----------------------------------------------------------------
6. TREE OF THOUGHT
----------------------------------------------------------------
Creeaza un prompt care cere explorarea a minimum 3 solutii diferite
pentru un plan de antrenament si alegerea celei mai bune, cu justificare.

Prompt:

Exploreaza minimum 3 strategii diferite de antrenament pentru cresterea fortei si alege varianta optima.

Context utilizator:
	•	Adult 28 ani
	•	Nivel intermediar
	•	3 zile/saptamana disponibile

Instructiuni:
	•	Compara avantajele si dezavantajele fiecarei strategii
	•	Alege solutia cea mai eficienta

Format de iesire:
	•	Strategie 1 (descriere + pro/contra)
	•	Strategie 2 (descriere + pro/contra)
	•	Strategie 3 (descriere + pro/contra)
	•	Concluzie + plan final

----------------------------------------------------------------
8. PROMPT CHAINING
----------------------------------------------------------------
Construieste un prompt cu doua etape clare:
1) analiza antrenament HIIT
2) generare plan structurat

Prompt:

Realizeaza sarcina in doua etape clare.

Context utilizator:
	•	Adult 32 ani
	•	Vrea antrenament HIIT pentru slabire

Instructiuni:
Etapa 1: Analizeaza beneficiile si riscurile antrenamentului HIIT.
Etapa 2: Creeaza un plan structurat de 4 saptamani.

Format de iesire:
	•	Sectiunea „Analiza"
	•	Sectiunea „Plan HIIT" cu structura pe saptamani


----------------------------------------------------------------
9. REACT (Reason + Act) + PERSONIFICARE
----------------------------------------------------------------
Scrie un prompt in care modelul explica rationamentul (Reason)
si propune actiuni concrete (Act), intr-un format clar
pentru un antrenament saptamanal.

Prompt:

Actioneaza ca un coach fitness experimentat si ofera raspuns in format Reason + Act.

Context utilizator:
	•	Adult 45 ani
	•	Vrea antrenament pentru sanatate generala

Instructiuni:
	•	Explica mai intai rationamentul (Reason)
	•	Apoi propune actiuni concrete (Act)

Format de iesire:
	•	Reason (analiza logica)
	•	Act (plan saptamanal clar)


----------------------------------------------------------------
10. MULTIMODAL PROMPTING + PERSONIFICARE
----------------------------------------------------------------
Formuleaza un prompt in care modelul este personificat ca
„antrenor intr-o sala de fitness", primeste o imagine
si recomanda exercitii pe baza ei.

Prompt:


Actioneaza ca un antrenor intr-o sala de fitness.

Vei primi o imagine cu o persoana care se antreneaza.

Context utilizator:
	•	Persoana incepatoare
	•	Vrea corectarea tehnicii si exercitii suplimentare

Instructiuni:
	•	Analizeaza postura si echipamentul din imagine
	•	Recomanda exercitii potrivite nivelului

Format de iesire:
	•	Observatii despre imagine (bullet points)
	•	Recomandari exercitii
	•	Sfaturi de siguranta
